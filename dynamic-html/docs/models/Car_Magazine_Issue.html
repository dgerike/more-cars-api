Car_Magazine_Issue

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The title of this issue or - if not available - the main topic.
  • number : Integer
    The issue's number as is printed on the cover. E.g. '9' for the September issue or '483' if the magazine is using consecutive numbering.
  • year : Integer
    The year that is printed on the cover. This might vary from the real release date, e.g. issue '1/2010' could already have been released in December of 2009.
  • release_date : date
    The official release date of the print version of this issue.
  • single_copy_price : Double
    The price for this issue on the domestic market, as printed on the cover, incl. VAT.
  • single_copy_price_unit : String
  • pages : Integer
    The total number of pages this issue has. This includes cover and flipside, but not booklets or flyers.
