Video

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The title of the video.
  • video_platform : String
    The platform or service that is hosting the video.
    Enum:
    youtube
  • video_platform_id : String
    The ID of the video, as given by the video hosting service.
  • source : String
    The URL of the video or the web page where the licensing and additional information can be found.
  • creator : String
    The name of the creator (or uploader) of the video.
  • license : String
    The license under which the video was published.
  • duration : String
    The length of this video, following the ISO 8601 standard.
  • description : String
  • tags : String
    A comma-separated list of keywords, describing the key aspects of this video.
