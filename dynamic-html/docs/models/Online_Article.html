Online_Article

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The headline of this article.
  • description : String
    Characterization of the article (Car review? News flash? Interview? Race report?) or an excerpt or a quote.
  • author : String
    The creator of the article.
  • publish_date : date
    The date when the article was published. Format is `YYYY-MM-DD`.
  • url : String
    Where can the article be found on the internet?
