Car_Model

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The official name of this model, respecting upper/lower case, whitespaces and special characters (e.g. \"cee'd\" or \"e-tron\"). The brand name is always omitted!
  • generation : Integer
    When this model was build over multiple generations, which one is this?
  • code : String
    The internal code of this model as given by the manufacturer.
  • built_from : Integer
    What was the first year of production for this model?
  • built_to : Integer
    What was the last year of production for this model?
  • total_production : Integer
    When adding up all variations, how many units of this model (of this generation) were produced in total?
