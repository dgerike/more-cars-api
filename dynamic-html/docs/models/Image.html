Image

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The name resp. title of this image.
  • image_platform : String
    The provider that is hosting the image.
    Enum:
    flickr
    wikimedia
  • image_platform_id : String
    The ID resp. the name of the image, as given by the image platform. Flickr uses numbers, like `43644091312`. Wikimedia uses a URL friendly version of the name of the image, like `File:BMW_Z4_Concept.jpg`.
  • source : String
    The internet address of the concrete image file.
  • origin : String
    A description page where additional information about the photo can be found. Like licensing details, author, camera model etc.
  • author : String
    The name of the photographer resp. the rights holder of the image.
  • license : String
    The license this image was published under.
  • description : String
    A short description of what can be seen on the photo, where it was taken, when it was taken.
  • tags : String
    A comma-separated list of keywords describing the picture.
