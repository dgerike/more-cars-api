Car_Programme_Episode

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The official name or title of this episode. Alternatively the main topic when there is no official title.
  • season : Integer
    The season number this episode belongs to.
  • season_episode : Integer
    The episode number within this season. (This is not the overall count!)
  • original_air_date : date
    The date this episode was broadcasted for the very first time.
  • duration : String
    The length of this episode in minutes and seconds.
