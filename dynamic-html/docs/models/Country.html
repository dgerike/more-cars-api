Country

  • id : Integer
    The internal `More Cars` ID of this node. It's generated by the system and cannot be altered.
  • name : String
    The official (english) name of the country.
  • adjectival : String
    The adjectival form of the country's name. E.g.: 'The `Italian` carmaker...' or 'The `British` version of this model...'
  • demonym : String
    How are the inhabitants of this country called? E.g.: 'The `Italians` have the fastest cars...')
  • short_name : String
    Short form of the country name (if applicable).
  • code : String
    The official (2-letter) country code, according to ISO 3166-1.
